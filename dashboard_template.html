Rolling multicollinearity (VIF)

VIF depends only on the regressor set, not on any one sector, so this is one chart for the whole model. A single latest-window VIF can look clean even if two regressors were highly collinear in an earlier regime; this tracks VIF through time so a past danger zone is not hidden. The dashed line marks the usual VIF = 5 concern threshold.
